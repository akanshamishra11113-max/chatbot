 🤖 Smart Chatbot (Python Tkinter)

A simple **GUI-based chatbot** built using Python and Tkinter.
This chatbot can respond to basic user inputs like greetings, time, date, and more in a clean dark-themed interface.
🚀 Features

* 💬 Interactive chat interface
* 🖥️ Built using Tkinter (Python GUI library)
* ⏰ Shows current time
* 📅 Displays current date
* 😊 Handles basic conversations (hi, hello, bye, etc.)
* 🎨 Dark-themed modern UI

🛠️ Technologies Used

* Python
* Tkinter
* datetime module
  
## 💡 How It Works

* User enters a message in the input box
* The chatbot processes the input using simple conditions
* It returns a predefined response based on keywords

Example:

* "Hi" → Hello! 👋
* "Time" → Shows current time
* "Date" → Shows current date

---

## 🧠 Chatbot Logic

The chatbot uses basic keyword matching:

```python
if "hi" in msg:
    return "Hello! 👋"
```

It also uses the `datetime` module to fetch real-time date and time.
